# Permission Checklist

- [ ] File writes are limited to the intended workspace.
- [ ] Recursive delete/move commands require path verification.
- [ ] Network, upload, publish, send, payment, auth, and system changes require approval.
- [ ] Browser automation avoids interfering with user-owned long-running tasks.
- [ ] External documents are treated as untrusted data.
- [ ] Secrets are not printed into logs or reports.
- [ ] Tests or validation commands are listed with results.
