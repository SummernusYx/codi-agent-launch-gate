# Security Policy

Agent Launch Gate is a review workflow, not a guarantee that an AI agent application is safe.

## What This Project Does

- Helps inspect AI agent app safety boundaries.
- Produces local evidence-backed scorecards and release decisions.
- Encourages confirmation gates before sensitive actions.
- Treats remote content as untrusted data.

## What This Project Does Not Do

- It does not replace dependency scanning.
- It does not replace secret scanning.
- It does not guarantee prompt-injection resistance.
- It does not certify production readiness.
- It does not upload code or scan remote services by default.

## Reporting Issues

If you find a security issue in this skill's instructions, templates, examples, or release process, open a private security advisory if the repository host supports it. Otherwise, open an issue without including secrets, tokens, private code, or exploit details that would put users at risk.

## Sensitive Data

Do not include API keys, cookies, tokens, private chat logs, customer data, or proprietary code in public reports.
